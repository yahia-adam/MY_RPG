 <h1 align="center"> MY_RPG </h1>

Create your own RPG

<h2>Requirements:</h2>
A Linux distribution or Apple's macOS Make GCC, CSFML 2.5

<h2>Compilation: </h2>
Navigate to the root of the repository from your Terminal and run make to build the my_rpg executable file.
make; ./my_rpg
